## 🎯 核心思路

**用 NotebookLM 逆向工程任何成功的 YouTube 频道，在几分钟内建立内容工厂。**

从 0 想法 → 全自动视频系统

## 🔄 7 步工作流（完整还原 + 优化）

### Step 1: 频道分析
```
Prompt:
"分析这个 YouTube 频道的成功模式：[频道链接]
- 最受欢迎的 10 个视频主题是什么？
- 标题结构有什么规律？
- 视频时长分布如何？
- 评论区观众最常问什么问题？
- 更新频率是多少？"
```

**输出：** 频道成功要素清单

---

### Step 2: 内容拆解
```
Prompt:
"选取这个频道播放量最高的 5 个视频，逐一分析：
1. 开场前 30 秒说了什么？（钩子）
2. 视频结构是怎样的？（几部分/章节）
3. 核心观点/价值点是什么？
4. 结尾如何引导互动？
5. 封面图有什么特点？"
```

**输出：** 爆款视频模板库

---

### Step 3: 选题生成
```
Prompt:
"基于以上分析，为我生成 30 个视频选题，要求：
- 符合该频道的内容风格
- 标题使用相同的结构模式
- 覆盖不同难度层级（新手/进阶/高级）
- 包含热点话题和长青话题
- 每个选题附带一句话描述"
```

**输出：** 30 天内容日历

---

### Step 4: 脚本创作
```
Prompt:
"为这个选题写一个完整的 YouTube 脚本：[选题]
要求：
- 开场钩子（前 30 秒，制造好奇/冲突/共鸣）
- 内容主体（分 3-5 个章节，每章节有小标题）
- 每 2-3 分钟设置一个互动点（提问/投票/评论引导）
- 结尾金句 + 行动号召（订阅/点赞/评论）
- 总时长控制在 [8-12] 分钟
- 语气风格：[参考频道风格]"
```

**输出：** 完整视频脚本

---

### Step 5: 标题优化
```
Prompt:
"为这个脚本生成 10 个 YouTube 标题，要求：
- 使用数字（如'7 个方法'、'3 步搞定'）
- 包含情绪词（惊人/秘密/终极/必看）
- 长度控制在 50-60 字符
- 包含目标关键词
- 避免标题党但要有吸引力
- 按点击率潜力排序"
```

**输出：** 标题 A/B 测试列表

---

### Step 6: 描述和标签
```
Prompt:
"为这个视频生成：
1. YouTube 描述（前 2 行包含核心关键词，包含时间戳章节）
2. 15 个相关标签（5 个核心词 +5 个长尾词 +5 个热点词）
3. 3 个置顶评论建议（引导互动/补充资源/问题收集）
4. 社区帖子文案（发布前预热）"
```

**输出：** 发布素材包

---

### Step 7: 批量生产
```
Prompt:
"基于这个工作流，帮我创建一个内容生产 SOP：
1. 每日工作流程（选题→脚本→录制→发布）
2. 批量创作方法（一次写 5-10 个脚本）
3. 复用策略（1 个核心内容→多平台改编）
4. 数据追踪表（播放量/完播率/互动率）
5. 优化迭代机制（每周复盘）"
```

**输出：** 内容工厂运营手册

## 📈 融入我们的内容工厂

### 现有架构升级

```
原流程：
Scout → Factory → Auditor → 发布

升级后：
Scout (情报) → NotebookLM 分析 → Factory (生产) → Auditor (审计) → 发布
```

### 新增能力

1. **YouTube 频道逆向工程**
   - 输入：任意 YouTube 频道链接
   - 输出：成功要素报告 + 内容模板

2. **批量脚本生成**
   - 输入：选题列表
   - 输出：完整脚本库（一次生成 10-20 个）

3. **多平台改编**
   - 输入：1 个 YouTube 脚本
   - 输出：抖音/小红书/B 站/公众号版本

## 💡 关键洞察

1. **不是复制内容，而是复制成功模式**
   - 学习结构，不抄袭创意
   - 保持自己的风格和观点

2. **数据驱动迭代**
   - 每个视频都是实验
   - 追踪完播率、互动率、收藏率

3. **规模化生产**
   - 一次分析，多次复用
   - 批量创作 > 单点创作
