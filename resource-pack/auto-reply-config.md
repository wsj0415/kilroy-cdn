## 🎯 运营技巧

### 提高回复率
1. **及时回复**: 30 分钟内回复
2. **个性化**: 加上用户昵称
3. **引导互动**: 问问题引发回复
4. **表情包**: 适当使用表情

### 提高转化率
1. **价值前置**: 先给价值再引导
2. **限时福利**: 制造紧迫感
3. **案例展示**: 用案例证明价值
4. **从众心理**: "已有 1000+ 人领取"

### 避免违规
1. **不直接发链接**: 用"私信获取"
2. **不频繁私信**: 避免被举报
3. **不用敏感词**: 如"赚钱""副业"
4. **引导到私域**: 微信/QQ 群
