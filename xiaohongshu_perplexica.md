## 🏠 部署超简单

**有 Docker 就行，一行命令搞定：**

```bash
docker run -d -p 3000:3000 \
  -v perplexica-data:/home/perplexica/data \
  --name perplexica \
  itzcrazykns1337/perplexica:latest
```

然后打开 `http://localhost:3000` 就能用了！

**支持的平台：**
- 💻 Windows / Mac / Linux
- ☁️ 云服务器（VPS）
- 🏠 NAS（群晖等）
