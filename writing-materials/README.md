## 📝 收录标准

### 钩子库
- 必须来自真实爆款内容（收藏率>1.5%）
- 标注完整来源（链接 + 作者 + 平台）
- 附带分析（为什么有效）

### 金句库
- 可直接引用或改编
- 标注出处
- 标注适用场景

### 案例库
- 真实故事或数据
- 有细节（时间/地点/人物/结果）
- 可复用的方法论

### 数据库
- 行业报告/研究数据
- 标注来源链接
- 标注发布日期（确保时效性）

### 禁词库
- AI 常用表达
- 提供替代方案
- 附带示例对比
