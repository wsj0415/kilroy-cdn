## 禁词库（AI 痕迹词）

### 2026-03-07 更新

| 禁词 | 替代方案 | 示例 |
|------|---------|------|
| 总之 | 说人话 / 直白点 | ❌"总之..." → ✅"说人话..." |
| 综上所述 | 说白了 | ❌"综上所述..." → ✅"说白了..." |
| 不仅如此 | 还有 / 另外 | ❌"不仅如此..." → ✅"还有..." |
| 关键在于 | 重点是 / 核心就一点 | ❌"关键在于..." → ✅"重点是..." |
| 首先/其次/最后 | 直接用 1/2/3 或者不用序号 | ❌"首先...其次...最后" → ✅ 直接说 |
| 希望对你有所帮助 | 今天就试试 / 有问题找我 | ❌"希望对你有所帮助" → ✅"今天就试试" |
| 如有疑问请留言 | 卡壳了？私信我 | ❌"如有疑问请留言" → ✅"卡壳了？私信我" |
| 太棒了 | （去掉，直接说事） | ❌"太棒了！" → ✅ 直接说结果 |
| 准备好了吗 | （去掉，直接开始） | ❌"准备好了吗？" → ✅ 直接开始 |
| ✅ ❌ 🎯 📊 | 少用或不用 emoji | 每 300 字≤3 个 |
