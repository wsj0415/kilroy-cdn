## 🎨 颜色规范

### Bento Grid 风格
- **主色调：** 科技蓝 (#2563EB)、Perplexity 绿 (#10B981)
- **辅助色：** 30-40% 饱和度的同色系
- **背景：** 浅色渐变 + 抽象科技图案（10-15% 透明度）

- **绿色区域：** #10B981 → #34D399（人类主导）
- **黄色区域：** #FBBF24 → #FCD34D（人机协作）
- **橙色区域：** #F97316 → #FB923C（AI 主导）
- **红色区域：** #EF4444 → #F87171（AI 独立）
