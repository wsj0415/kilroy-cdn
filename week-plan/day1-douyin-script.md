### 【15-45 秒】核心教程 - 分步演示
**画面**: 终端录屏（清晰展示命令）

**文案**（配合操作步骤）:

**Step 1** (15-22 秒):
> "第一步，创建同步脚本"
> "在 skills 目录下新建 sync-skills.sh"

**终端显示**:
```bash
cd /root/.openclaw/agents/content/skills
nano sync-skills.sh
```

**Step 2** (22-30 秒):
> "第二步，添加同步命令"
> "用 rsync 自动同步两个目录"

**终端显示**:
```bash
rsync -av --exclude='.git' \
  /root/.openclaw/agents/content/skills/ \
  /root/.openclaw/workspace-kilroy-content/skills/
```

**Step 3** (30-38 秒):
> "第三步，执行同步"
> "一行命令，搞定！"

**终端显示**:
```bash
./sync-skills.sh --to-kilroy
# 输出：✅ 同步完成！
```

**Step 4** (38-45 秒):
> "还可以反向同步"
> "双向同步，随心所欲"

**终端显示**:
```bash
./sync-skills.sh --to-content   # 反向
./sync-skills.sh --list         # 查看技能列表
```
