# Social Content Generator - 社媒内容生成器

**一句话**：链接 → 完整内容包（文案 + 标签 + 封面图 + 标题）

---

## 🚀 快速开始

```
用户提供链接 → AI 抓取内容 → 生成完整内容包
```

### 基础使用

```
用户：https://example.com/article

AI: 自动生成：
1. 推文文案（3 个版本）
2. SEO 标签（按平台分类）
3. 系列封面图提示词（3 张一套）
4. 吸睛标题（5 个选项）
5. 一句话描述（3 个版本）
```

### 自定义风格

```
用户：https://example.com/article --style=glassmorphism --platform=xiaohongshu

AI: 根据指定风格和平台生成内容
```

## 📋 完整工作流

### Step 1: 链接抓取与分析

**工具**: `web_fetch` 或 `browser`

**提取内容**:
- 页面标题
- 核心内容摘要
- 关键数据/亮点
- 目标受众
- 内容类型（工具/教程/观点/产品）

### Step 2: 生成推文文案

**3 个版本**:
1. **钩子版** - 强吸引力开头
2. **干货版** - 信息密度高
3. **故事版** - 情绪共鸣强

**结构**:
```
钩子 (3 秒抓住注意力)
↓
痛点/价值 (为什么重要)
↓
核心内容 (3-5 个亮点)
↓
行动号召 (评论/收藏/转发)
↓
标签 (5-10 个)
```

### Step 3: SEO 标签推荐

**分类提供**:
- 流量型标签（高搜索量）
- 垂类标签（精准受众）
- 平台专属标签
- 品牌标签

**按平台优化**:
| 平台 | 数量 | 位置 |
|------|------|------|
| Twitter/X | 3-5 个 | 文末 |
| 小红书 | 8-12 个 | 文末 |
| 即刻 | 5-8 个 | 文末 |
| LinkedIn | 3-5 个 | 文末 |

### Step 4: 系列封面图设计

**默认**: 3 张一套，讲述完整故事

**故事线**:
1. **封面 1 - 问题篇**: 痛点共鸣 😰
2. **封面 2 - 方案篇**: 产品亮相 😃
3. **封面 3 - 行动篇**: 转化推动 🚀

**可选风格**:

| 风格 | 代码 | 说明 |
|------|------|------|
| 设计工具风 | `design-tool` | Figma 紫/粉渐变，Glassmorphism |
| 极客代码风 | `geek-code` | Terminal 深色，代码高亮 |
| 极简卡片风 | `minimal-card` | 黑白灰，瑞士设计 |
| 温暖生活风 | `warm-life` | 暖橙渐变，生活场景 |
| 赛博朋克风 | `cyberpunk` | 霓虹蓝紫，未来感 |
| 商务专业风 | `business-pro` | 深蓝金边，高端质感 |

**输出格式**:
```markdown
### 封面 1: 问题篇

**提示词**:
[完整的 AI 绘图提示词，包含构图、配色、文字位置]

**情绪**: 焦虑/困扰
**核心信息**: "你有这个问题"

---

### 封面 2: 方案篇

**提示词**:
[完整的 AI 绘图提示词]

**情绪**: 希望/期待
**核心信息**: "我能解决它"

---

### 封面 3: 行动篇

**提示词**:
[完整的 AI 绘图提示词]

**情绪**: 兴奋/行动
**核心信息**: "现在就试试"
```

### Step 5: 吸睛标题推荐

**5 个版本**,涵盖不同类型:

1. **数字型** - "3 个步骤..."
2. **疑问型** - "为什么你总是..."
3. **对比型** - "月薪 3000 vs 月薪 3 万..."
4. **警告型** - "千万别..."
5. **结果型** - "靠这个副业，我..."

### Step 6: 一句话描述

**3 个版本**:
1. **功能导向** - 直接说明是什么
2. **利益导向** - 强调用户收益
3. **情绪导向** - 引发情感共鸣

## 🎨 视觉风格详解

### 设计工具风 (design-tool)

```
主色：Figma 紫 (#7c3aed) → 粉 (#ec4899) 渐变
元素：Glassmorphism 卡片、漂浮组件、设计工具 UI
字体：几何无衬线体
适用：设计师、创意工具、UI/UX 产品
```

### 极客代码风 (geek-code)

```
主色：GitHub 深 (#0d1117) + 青 (#58a6ff) + 绿 (#238636)
元素：Terminal 窗口、代码高亮、命令提示符
字体：等宽字体
适用：开发者工具、API、技术产品
```

### 极简卡片风 (minimal-card)

```
主色：黑白灰 (#1a1a1a, #666666, #f5f5f5)
元素：圆角卡片、柔和阴影、留白
字体：超粗无衬线 + 细体对比
适用：高端产品、咨询服务、知识付费
```

### 温暖生活风 (warm-life)

```
主色：暖橙 (#ff6b35) → 桃 (#f7c59f) 渐变
元素：生活场景、自然光线、纸质纹理
字体：圆体/手写体
适用：生活方式、个人成长、副业变现
```

### 赛博朋克风 (cyberpunk)

```
主色：霓虹蓝 (#00f3ff) + 紫 (#bc13fe) + 黑
元素：光效、电路纹理、未来 UI
字体：科技感字体
适用：AI 产品、区块链、前沿科技
```

### 商务专业风 (business-pro)

```
主色：深蓝 (#1e3a5f) + 金 (#d4af37)
元素：几何线条、金属质感、高端纹理
字体：衬线体 + 无衬线体组合
适用：企业服务、金融、咨询
```

## 📱 平台适配规则

### Twitter/X

- **文案长度**: 280 字符内
- **标签数量**: 3-5 个
- **封面比例**: 16:9 或 1:1
- **风格推荐**: 极客代码、商务专业

### 小红书

- **文案长度**: 300-800 字
- **标签数量**: 8-12 个
- **封面比例**: 3:4 或 9:16
- **风格推荐**: 设计工具、温暖生活

### 即刻

- **文案长度**: 200-500 字
- **标签数量**: 5-8 个
- **封面比例**: 1:1 或 4:3
- **风格推荐**: 极简卡片、设计工具

### LinkedIn

- **文案长度**: 300-1000 字
- **标签数量**: 3-5 个
- **封面比例**: 1.91:1 或 1:1
- **风格推荐**: 商务专业、极简卡片

## 🎯 最佳实践

### 钩子公式

```
数字冲击："3 个步骤..."
情绪挑衅："听我一句劝..."
悬念制造："我发现了一个..."
身份标签："打工人必看..."
结果承诺："月入 3 万..."
```

### 封面设计原则

1. **文字不超 3 行** - 移动端可读性
2. **对比度>4.5:1** - 可访问性
3. **安全区域** - 避开平台 UI 遮挡
4. **品牌一致性** - 3 张封面视觉 DNA 统一

### 标签策略

```
2 个品牌标签 + 4 个垂类标签 + 4 个流量标签 + 2 个工具标签
```
