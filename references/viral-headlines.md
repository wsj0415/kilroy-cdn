# 爆款标题模板库

## 数字型标题

```
[数字] + [身份/人群] + [痛点/结果]
```

**示例**:
- 30 岁前一定要知道的 5 个理财真相
- 我用 3 个方法，3 个月瘦了 20 斤
- 95% 的人都不知道的 iPhone 隐藏功能
- 3 个步骤，让你的效率翻倍
- 5 款神器，打工人必备

**适用场景**: 教程、资源推荐、方法论

---

## 疑问型标题

```
为什么 + [人群] + [痛点]?
如何 + [达成目标]?
[问题] 的真相是什么？
```

**示例**:
- 为什么你总是存不下钱？
- 如何判断一个男人值不值得嫁？
- 什么样的人适合做自媒体？
- 为什么你的推文没人看？
- 如何月入过万？

**适用场景**: 观点文、经验分享、问题解答

---

## 对比型标题

```
[状态 A] vs [状态 B] 的差距
[做 X] 之前 vs 之后
[方法 A] vs [方法 B]，我选了后者
```

**示例**:
- 月薪 3000 和月薪 3 万的思维差距
- 做自媒体前 vs 做自媒体后
- 报课学习 vs 自学，我选了后者
- 用 AI 前 vs 用 AI 后，效率差了 10 倍

**适用场景**: 案例分享、产品对比、成长记录

---

## 警告型标题

```
千万别 + [行为]，会 [后果]
这 [数字] 个 [坑]，我替你踩过了
不要再 + [无效行为] 了
```

**示例**:
- 千万别这样发小红书，会限流
- 这 5 个副业坑，我替你踩过了
- 不要再无效努力了
- 听我一句劝，别再去报那些 1999 的课了

**适用场景**: 避坑指南、行业揭秘、劝诫文

---

## 结果型标题

```
靠这个 + [方法/工具]，我 [成果]
用这个方法，我 [时间] [成果]
学会这个技能，[收益]
```

**示例**:
- 靠这个副业，我月入 5 万
- 用这个方法，我 3 天涨粉 1000
- 学会这个技能，薪资翻倍
- 靠 AI 工具，我每天只工作 2 小时

**适用场景**: 成果展示、案例分享、变现故事

---

## 身份标签型

```
[身份] 必看
[年龄段] 一定要知道
[职业] 的 [数量] 个真相
```

**示例**:
- 打工人必看
- 30 岁前一定要知道的 10 个道理
- 产品经理的 5 个潜规则
- 独立开发者必须知道的 3 件事

**适用场景**: 人群定向、身份认同

## 悬念制造型

```
我发现了一个 [形容词] 的 [事物]
[时间] 后，我发现了这个秘密
[行为] 的真相，第 [数字] 个颠覆认知
```

**示例**:
- 我发现了一个极其阴险的推流逻辑
- 做自媒体 3 年后，我发现了这个秘密
- 关于 AI 的 5 个真相，第 3 个颠覆认知
- 凌晨 3 点，我发现了这个暴利玩法

**适用场景**: 揭秘文、行业洞察、深度分析

---

## 情绪共鸣型

```
[场景]，我差点 [情绪行为]
[时间] 的 [经历]，让我明白了 [道理]
致 [人群]: [情感表达]
```

**示例**:
- 凌晨三点，看着后台数据，我差点砸了电脑
- 35 岁的裁员通知，让我明白了这个道理
- 致所有打工人：你值得更好的

**适用场景**: 故事文、情感文、成长感悟

## 使用指南

### 选择原则

| 内容类型 | 推荐标题类型 | 转化率 |
|---------|------------|--------|
| 教程/资源 | 数字型 | ⭐⭐⭐⭐⭐ |
| 观点/洞察 | 疑问型/悬念型 | ⭐⭐⭐⭐ |
| 案例/成果 | 结果型/对比型 | ⭐⭐⭐⭐⭐ |
| 避坑/揭秘 | 警告型 | ⭐⭐⭐⭐⭐ |
| 故事/情感 | 情绪共鸣型 | ⭐⭐⭐⭐ |

### A/B 测试建议

同时准备 3-5 个标题，测试:
1. **点击率** - 哪个标题更吸引人
2. **完读率** - 哪个标题与内容匹配度高
3. **互动率** - 哪个标题引发更多评论

### 标题优化检查清单

- [ ] 包含数字（具体、可感知）
- [ ] 明确目标人群
- [ ] 击中痛点或承诺结果
- [ ] 长度适中（15-25 字最佳）
- [ ] 无错别字和语病
- [ ] 符合平台调性
