# 科技产品摄影提示词模板

## 适用场景
- 高端、专业、商业感
- B 站/YouTube 封面
- 视觉冲击力强

## 基础提示词

```prompt
Ultra-realistic 3D commercial-style product shot of [产品描述].

Subject: [产品名称] - [一句话描述]

Composition:
- Center: [核心视觉元素]
- Surrounding elements:
  * [元素 1]
  * [元素 2]
  * [元素 3]

Lighting:
- Luxury cinematic studio lighting
- Dramatic rim lighting to define [物体] edges
- Brilliant highlights on [高光部分]
- Soft [颜色] backlight for depth and richness

Color Palette:
- Background: [背景色描述]
- Accents: [强调色]
- Interface: [界面色]

Technical Specs:
- Camera: [镜头描述]
- Depth of field: [景深描述]
- Quality: 8K resolution, ultra-photorealistic texture
- Aspect ratio: [尺寸比例]

Output: 1 image, [风格描述].
```

## 示例（Skills Manager）

```prompt
Ultra-realistic 3D commercial-style product shot of a premium software tool.

Subject: Skills Manager - Universal AI Agent Skills Management Platform

Composition:
- Center: Floating holographic interface showing 11 AI tool icons (Claude Code, Cursor, GitHub Copilot, etc.)
- Surrounding elements: 
  * Code snippets floating in mid-air
  * Skill modules as glowing cubes
  * Connection lines between tools (neural network style)
  * GitHub logo and open source badge

Lighting:
- Luxury cinematic studio lighting
- Dramatic rim lighting to define interface edges
- Brilliant highlights on holographic elements
- Soft blue backlight for depth and richness

Color Palette:
- Background: Deep charcoal fading into warm amber glow
- Accents: Electric blue, cyan highlights, gold GitHub accents
- Interface: Translucent glass with blue tint

Technical Specs:
- Camera: Macro lens, slight low angle for dominance
- Depth of field: Shallow focus on center interface
- Quality: 8K resolution, ultra-photorealistic texture
- Aspect ratio: 16:9 landscape

Output: 1 image, tech product photography style, open source software visualization.
```
