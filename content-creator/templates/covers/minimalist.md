# 极简科技海报提示词模板

## 适用场景
- 社交媒体传播
- 视觉冲击力强
- 小红书/Instagram

## 基础提示词

```prompt
Minimalist tech poster design for [产品名称].

Concept: "[核心理念]"

Visual Elements:
- Center: [核心视觉]
- Arrangement: [布局方式]
- Each element represented by [表现方式]

Style:
- Flat design with subtle gradients
- Clean geometric shapes
- Abundant white space
- Modern sans-serif typography

Color Scheme:
- Background: [背景色]
- [主元素]: [颜色描述]
- Text: [文字颜色]

Composition:
- Rule of thirds
- [布局规则]
- Clean margins
- Magazine-quality production

Technical:
- Aspect ratio: [尺寸比例]
- Resolution: Ultra HD
- Style: [风格描述]

Output: 1 image, [最终描述].
```

## 示例（Skills Manager）

```prompt
Minimalist tech poster design for Skills Manager.

Concept: "One Tool to Rule Them All"

Visual Elements:
- Center: Large circular hub with "Skills Manager" text
- Radiating from center: 11 clean lines connecting to AI tool logos
- Arrangement: Symmetrical radial pattern (like a constellation map)
- Each tool represented by its iconic logo/symbol

Style:
- Flat design with subtle gradients
- Clean geometric shapes
- Abundant white space
- Modern sans-serif typography

Color Scheme:
- Background: Pure white or very light gray
- Hub: Gradient from blue to purple
- Lines: Thin, light gray
- Tool logos: Their official brand colors
- Text: Dark charcoal (#1F2937)

Composition:
- Rule of thirds
- Center-weighted design
- Clean margins
- Magazine-quality production

Technical:
- Aspect ratio: 16:9 landscape
- Resolution: Ultra HD
- Style: Modern tech poster, Apple keynote aesthetic

Output: 1 image, minimalist tech poster, clean and professional.
```
