# Content Creator - 智能内容创作助手

## 技能描述

多平台内容创作技能，支持**一步步引导用户**完成内容创作全流程：
1. 需求确认（风格/平台/尺寸）
2. 文案生成（多模板选择）
3. 封面图提示词生成
4. 发布建议

## 核心功能

- ✅ **风格选择**：提供多种文案风格供用户选择
- ✅ **平台适配**：根据目标平台自动调整文案格式
- ✅ **封面图生成**：Nano Banana Pro 提示词推荐
- ✅ **模板索引**：内置多套模板可检索
- ✅ **逐步引导**：交互式完成创作流程

## 工作流程

### Step 1: 需求收集（必须）

**收到用户创作请求后，先询问以下信息：**

```markdown
收到！🤖 在开始创作前，我需要确认几个问题：

## 📝 内容信息
1. **内容主题是什么？**
   - 开源项目介绍
   - AI 工具推荐
   - 技术教程
   - 观点输出
   - 案例分享
   - 其他（请说明）

2. **目标平台是？**
   - 微头条（300-500 字）
   - 抖音（60 秒脚本）
   - 小红书（图文 6-9 张）
   - B 站（8-10 分钟视频）
   - 公众号（1500+ 字文章）
   - 多平台分发

3. **文案风格偏好？**
   - 资讯分享型（客观介绍，不编造体验）
   - 实战测评型（真实使用体验 + 优缺点）
   - 痛点解决型（从痛点引出方案）
   - 对比评测型（多工具对比）
   - 教程型（手把手教学）
   - 让助手推荐

4. **需要封面图吗？**
   - 需要（我会生成 Nano Banana Pro 提示词）
   - 不需要

5. **封面图尺寸？**
   - 16:9（横版，适合微头条/B 站）
   - 9:16（竖版，适合抖音/小红书）
   - 1:1（方版，适合公众号）
   - 让助手推荐

---
**你可以直接回复序号，例如："1, 微头条，资讯分享型，需要，16:9"**
```

**根据用户选择的目标平台，自动调整：**

| 微头条 | 300-500 字 | 每 2-3 句换行 | 10 个 | 结论先行 |
| 抖音 | 60 秒脚本 | 分镜头格式 | 8-10 个 | 前 3 秒钩子 |
| 小红书 | 800-1200 字 | emoji+ 分段 | 10-15 个 | 封面吸睛 |
| B 站 | 1500+ 字 | 章节划分 | 8 个 | 前 30 秒留人 |
| 公众号 | 1500-3000 字 | 深度内容 | 5-8 个 | 标题党 |

## 使用指南

### 基本用法

```bash
# 交互式创作（推荐）
python3 scripts/content_creator.py

# 指定参数创作
python3 scripts/content_creator.py \
  --topic "Skills Manager" \
  --platform weitoutiao \
  --style news_share \
  --cover true \
  --cover-ratio 16:9
```

### 与其他技能协作

- **nano-banana-pro**：生成封面图提示词
- **x-fetcher**：抓取 X 推文内容
- **link-summarizer**：总结网页内容
- **youtube-subtitle**：提取 YouTube 字幕

## 核心原则

### ✅ 必须遵守
1. **实事求是**：没用过的不说，不编造体验
2. **逐步引导**：先确认需求再生成内容
3. **模板索引**：提供多套模板供选择
4. **平台适配**：根据平台调整格式

### ❌ 禁止行为
1. 编造"实测体验"或"使用时长"
2. 编造"坑"或"缺点"
3. 不确认需求就直接生成
4. 忽略平台格式要求
